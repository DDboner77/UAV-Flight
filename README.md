## 🚀 项目计划

### 项目目标
在 **2周内** 实现基于凌霄飞控的无人机稳定起降功能，搭建350mm轴距碳纤维无人机平台，为后续导航算法开发奠定硬件基础。

同步关注路径规划赛题。
